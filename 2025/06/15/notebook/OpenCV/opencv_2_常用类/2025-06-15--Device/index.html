cv::ocl::Device 类 详解

cv::ocl::Device 是 OpenCV 对 OpenCL 中 cl_device_id 的封装，属于 cv::ocl 命名空间，用于描述 一个 OpenCL 设备（如 GPU / CPU / 加速器） 的各种信息与功能。

✅ 作用

  • 封装一个 OpenCL 设备
  • 查询设备信息（名称、类型、内存大小、OpenCL 版本等）
  • 判断设备能力（是否支持 FP16 / 双精度 / 图像等）
  • 设置当前活跃设备（影响 OpenCL kernel 调用）

🔍 常用函数说明（含返回类型）

函数名 功能 示例
name() 设备名 "NVIDIA GeForce GTX 1060"
vendorName() 厂商名 "NVIDIA Corporation"
version() OpenCL 版本 "OpenCL 1.2 CUDA"
openclCVersion() OpenCL C 支持版本 "OpenCL C 1.2"
type() 返回设备类型（GPU/CPU） Device::TYPE_GPU
globalMemSize() 全局内存字节数 4294967296（4 GB）
localMemSize() 本地（共享）内存大小 通常为 32 KB / 64 KB
maxComputeUnits() 最大计算单元数（CU） 通常 8~80
maxWorkGroupSize() 最大 work group size 常见为 256/512/1024
maxWorkItemSizes() 每维最大工作项数 返回 Vec3<size_t>
isUnifiedMemory() 是否共享内存架构 true 表示 CPU 与 GPU 共享地址空间
hostUnifiedMemory() 同上，是否共享内存地址

📦 高级能力判断

函数 判断支持能力
doubleFPConfig() 是否支持双精度
halfFPConfig() 是否支持半精度
imageSupport() 是否支持 OpenCL 图像对象
nativeKernel() 是否支持 Native Kernel
extensions() 返回支持的扩展字符串（如 cl_khr_fp16）
available() 当前设备是否可用
