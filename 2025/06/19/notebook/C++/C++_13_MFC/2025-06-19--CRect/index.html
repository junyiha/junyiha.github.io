MFC CRect 详解

在 MFC 中，CRect 是一个非常常用的矩形类，广泛用于窗口布局、绘图、坐标计算等场景。它是 CPoint 和 CSize 的组合体，并继承自 tagRECT（Win32 的 RECT 结构），使它可以和 Win32 API 无缝配合。

🧱 一、定义与本质

CRect 实质上就是一个四元整数结构，表示 left, top, right, bottom。

📐 二、核心成员

成员变量 含义
left 左上角横坐标（x）
top 左上角纵坐标（y）
right 右下角横坐标（x）
bottom 右下角纵坐标（y）

🛠️ 三、常用方法

📏 尺寸与位置

函数 说明
int Width() 返回宽度：right - left
int Height() 返回高度：bottom - top
CPoint TopLeft() 返回左上角点
CPoint BottomRight() 返回右下角点
CPoint CenterPoint() 返回中心点
void SetRect(x1,y1,x2,y2) 设置矩形坐标
void SetRectEmpty() 清空为 (0,0,0,0)
BOOL IsRectEmpty() 是否是空矩形
BOOL PtInRect(POINT pt) 点是否在矩形内

🔧 偏移/缩放

函数 说明
void OffsetRect(dx, dy) 整体平移矩形
void InflateRect(dx, dy) 增大宽高，向外扩展
void DeflateRect(dx, dy) 缩小宽高，向内收缩
void NormalizeRect() 交换 top/bottom 或 left/right 保证合法

📐 交集/并集

函数 说明
BOOL IntersectRect(LPCRECT rect1, LPCRECT rect2) 当前对象 = 两矩形交集
BOOL UnionRect(LPCRECT rect1, LPCRECT rect2) 当前对象 = 两矩形并集

⚙️ 运算符重载

运算符 含义
operator == 比较两个矩形是否相等
operator += 平移
operator -= 平移反方向
operator & 返回交集
`operator ` 返回并集

🧵 五、与其他 MFC 类的关系

  • CRect ⊂ CDC：设备上下文中绘图常用 CRect 表示区域（如 DrawText, FillRect, Rectangle 等）
  • CRectTracker：用于图形编辑控件拖拉选中区域
  • CWnd::GetClientRect(CRect&)：获取窗口客户区大小

✅ 小结

优点 说明
Win32 API兼容性强 继承自 RECT，可直接传给 API 函数
功能丰富 支持平移、缩放、交集、并集等操作
用法灵活 与 CPoint、CSize、CDC 搭配使用顺畅

如你在做 控件布局、窗口坐标转换、绘图区域定义、HitTest 检测 等任务时，CRect 是非常好用的基础类之一。
