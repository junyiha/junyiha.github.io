Qt QMenuBar类 详解

Qt QMenuBar 类详解

QMenuBar 是 Qt 中用于创建菜单栏（Menu Bar）的类，通常用于 QMainWindow 作为应用程序的全局菜单。它包含多个 QMenu，每个 QMenu 内可以包含多个 QAction 作为菜单项。

📌 1. 基本用法

✅ 创建菜单栏并添加菜单

📝 代码解析

  • QMenuBar 负责创建菜单栏。
  • QMenu 用于创建下拉菜单，如 **”文件”、“编辑”**。
  • QAction 是菜单项，如 **”打开”、“退出”**。
  • addSeparator() 添加分隔线。
  • setMenuBar(menuBar) 设置主窗口的菜单栏。

📌 2. QMenuBar 的主要 API

✅ 2.1 添加菜单

  • &F 是快捷键 Alt+F 访问菜单。

✅ 2.2 添加菜单项

  • &N 是快捷键 Alt+N 访问 **”新建”**。

✅ 2.3 添加子菜单

  • 这样 “导出” 变成了 子菜单。

✅ 2.4 连接菜单项的槽函数

  • 当点击 “新建” 时，会触发 newFile() 函数。

✅ 2.5 删除菜单

  • removeAction() 可以删除菜单。

📌 3. 使用快捷键

✅ 3.1 通过 & 定义快捷键

  • Alt + S 触发 **”保存”**。

✅ 3.2 通过 QAction::setShortcut() 设置快捷键

  • Ctrl + S 触发 **”保存”**。

📌 5. Qt Designer 方式

  2. 拖动 QMenuBar 到窗口顶部。
  3. 添加 QMenu（菜单）和 QAction（菜单项）。

🔥 总结

✅ QMenuBar 用于创建菜单栏。
✅ QMenu 用于创建下拉菜单。
✅ QAction 是菜单项，可连接槽函数。
✅ setShortcut() 设定快捷键。
✅ 适用于 QMainWindow，但 QWidget 也可使用。

QMenuBar 是 Qt 应用程序中 菜单栏 的核心组件，掌握它可以让你的界面更加专业！🚀
