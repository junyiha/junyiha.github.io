8. 支持的压缩类型

  • zipfile.ZIP_STORED：不压缩。
  • zipfile.ZIP_DEFLATED：标准压缩（需要 zlib 支持）。
  • zipfile.ZIP_BZIP2：bzip2 压缩（需要 bz2 支持）。
  • zipfile.ZIP_LZMA：LZMA 压缩（需要 lzma 支持）。
